MCP Server Example

This is a barebones MCP (Model Context Protocol) server deployed on Netlify.

How to Use

This MCP server is available at /mcp. You can use it with any MCP client.

For example, with the MCP inspector:

npx @modelcontextprotocol/inspector npx mcp-remote@next https://your-site-name.netlify.app/mcp

Or add it to your Claude Desktop configuration:

{
  "mcpServers": {
    "my-mcp": {
      "command": "npx",
      "args": [
        "mcp-remote@next",
        "https://your-site-name.netlify.app/mcp"
      ]
    }
  }
}

Available Features

  • Run Analysis Report - A simple tool that generates sample data analysis
  • Interpreting Reports - A resource providing context on how to interpret reports

This is a minimal example that you can extend with your own tools and resources.
